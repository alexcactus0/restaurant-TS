# Restaurant Page (TypeScript Version)

## Overview  
This is a restaurant website project rebuilt using TypeScript. The original version was written in JavaScript, and this version was created to practice and improve TypeScript skills in a real project.

The project uses Vite as a build tool and follows a modular structure with TypeScript features like classes, interfaces, type aliases, and unions.

## Features  
- Restaurant homepage layout  
- Modular code structure  
- TypeScript-based logic  
- Component-style classes  
- Clean and reusable code organization  
- Vite development setup  

## Technologies Used  
- TypeScript  
- JavaScript (ES6+)  
- Vite  
- HTML  
- CSS  

## Concepts Practiced  
- TypeScript basics and advanced types  
- Object-oriented programming with classes  
- Interfaces and type aliases  
- Module-based project structure  
- Vite project setup and bundling  

## Purpose  
The purpose of this project was to rebuild an existing JavaScript project using TypeScript to improve code structure, type safety, and overall development skills.
